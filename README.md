# Mini WhatsApp CRUD App

A WhatsApp-inspired CRUD application built using Node.js, Express.js, EJS, and MongoDB Atlas.

## 🚀 Live Demo

https://mini-whatsapp-crud.onrender.com/chats

## 📌 Features

* View all chats
* Create new chats
* Edit existing messages
* Delete chats
* MongoDB Atlas cloud database integration
* Deployed on Render

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* EJS
* CSS
* Method Override

## 📂 Project Structure

```text
models/
views/
public/
index.js
init.js
```

## ⚙️ Installation

```bash
git clone https://github.com/mdjyaulhaque/mini-whatsapp-crud.git
cd mini-whatsapp-crud
npm install
```

Create a `.env` file:

```env
MONGO_URL=your_mongodb_connection_string
```

Run the application:

```bash
node index.js
```

Open:

```text
http://localhost:3000/chats
```
